Estruturas Condicionais

Introdução

As estruturas condicionais são um dos principais blocos da lógica de programação. Elas permitem que o programa tome decisões durante a execução, escolhendo diferentes caminhos de acordo com as condições verificadas.

Por exemplo: se um usuário estiver logado, o sistema exibe um menu personalizado; caso contrário, exibe a tela de login. Com as condicionais, podemos controlar o comportamento dos nossos programas.

Estrutura if / else

A estrutura if verifica se uma condição é verdadeira. Se for, o código dentro do bloco { } será executado:

if (idade >= 18) { console.log("Você é maior de idade."); }

O else permite executar um código alternativo quando a condição não for satisfeita:

if (idade >= 18) { console.log("Você é maior de idade."); } else { console.log("Você é menor de idade."); }

Podemos usar else if para testar múltiplas condições de forma sequencial:

if (nota >= 9) { console.log("Excelente!"); } else if (nota >= 7) { console.log("Aprovado"); } else if (nota >= 5) { console.log("Recuperação"); } else { console.log("Reprovado"); }

Condições compostas e operadores

Uma condição pode ser simples ou composta. Para compor condições, usamos operadores relacionais e lógicos:

  • == Igual a
  • != Diferente de
  • >, <, >=, <= Comparações numéricas
  • && (AND) — verdadeiro se ambas as condições forem verdadeiras
  • || (OR) — verdadeiro se pelo menos uma condição for verdadeira
  • ! (NOT) — inverte o valor lógico (verdadeiro vira falso e vice-versa)

Exemplo com condição composta:

if (idade >= 18 && idade < 60) { console.log("Adulto em idade ativa."); }

Outro exemplo com OR:

if (dia == "sábado" || dia == "domingo") { console.log("Fim de semana!"); }

E com NOT:

if (!usuarioLogado) { console.log("Por favor, faça login."); }

Estrutura switch / case

A estrutura switch é usada quando queremos comparar um valor com várias possibilidades fixas:

switch (opcao) { case 1: console.log("Nova partida."); break; case 2: console.log("Carregar jogo."); break; case 3: console.log("Configurações."); break; default: console.log("Opção inválida."); break; }

O break evita que o programa continue executando os próximos case após encontrar um correspondente.

O default é executado caso nenhuma das opções corresponda ao valor da variável.

If x Switch — quando usar cada um?

Use if / else quando:

  • As condições envolvem comparações complexas ou intervalos
  • Você precisa usar operadores lógicos (AND, OR, NOT)
  • As condições não são valores fixos

Use switch quando:

  • Está verificando um valor fixo contra várias opções
  • Quer deixar o código mais claro em situações com muitos if / else aninhados

If e Switch aninhados

Você pode colocar um if dentro de outro if, ou usar switch dentro de um bloco condicional. Isso se chama aninhamento:

if (usuarioLogado) { if (permissaoAdmin) { console.log("Acesso total."); } else { console.log("Acesso limitado."); } }

Exemplo de switch aninhado:

if (usuarioLogado) { switch (menu) { case 1: console.log("Perfil do usuário."); break; case 2: console.log("Configurações."); break; default: console.log("Opção inválida."); break; } }

Conceitos Avançados e Dicas

Aqui vão algumas dicas importantes para você dominar as estruturas condicionais:

Associação do else

Quando temos if aninhados sem chaves {}, o else sempre se associa ao último if não fechado:

if (x > 5) if (y < 3) console.log("A"); else console.log("B");

Execução sem chaves {}

Se não usar chaves, o if ou else executará apenas a primeira linha seguinte:

if (condicao) console.log("Linha 1"); console.log("Linha 2");

Switch sem break — comportamento em cascata

Se você esquecer o break em um case, a execução "cai" nos próximos cases:

switch (opcao) { case 1: console.log("Opção 1"); case 2: console.log("Opção 2"); default: console.log("Padrão"); }

Ordem de execução de && e ||

Em uma expressão como:

if (x > 5 && y < 10 || z == 20)

A ordem segue a precedência de operadores:

  1. O && (AND) é avaliado antes do || (OR).
  2. Use parênteses para tornar claro:
if ( (x > 5 && y < 10) || z == 20 )

Switch com múltiplos case sem código

switch (dia) { case "sábado": case "domingo": console.log("Final de semana!"); break; default: console.log("Dia útil"); }

Resumo Final e Boas Práticas

  • Use if e else para condições simples e complexas.
  • Use switch para valores fixos com múltiplas opções.
  • Combine operadores lógicos && e || para expressões compostas.
  • Evite esquecer break em switch (para não ter efeito cascata indesejado).
  • Use chaves {} mesmo em blocos de uma linha para evitar ambiguidades.
  • Evite aninhamentos excessivos — modularize com funções quando necessário.
  • Entenda a ordem de precedência de operadores para criar condições claras.
  • Pratique com casos reais (menus, validação de login, regras de jogo, etc).

Conclusão

O uso de if, else e switch é onipresente em qualquer linguagem de programação. Estas ferramentas são a base para criar softwares que "pensam" e respondem a diferentes situações.
