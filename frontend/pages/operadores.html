Operadores e Expressões

Introdução

Operadores e expressões são componentes fundamentais de qualquer algoritmo ou linguagem de programação. Eles permitem realizar cálculos, comparar valores e controlar o fluxo de execução de programas. Uma expressão combina valores, variáveis, operadores e funções, e seu resultado pode ser um número, um texto ou um valor lógico (verdadeiro ou falso).

Os operadores são classificados em três categorias principais:

  • Operadores Aritméticos
  • Operadores Relacionais (Comparação)
  • Operadores Lógicos

Operadores Aritméticos

Os operadores aritméticos executam operações matemáticas. Podem ser:

  • Unários — agem sobre um único operando (ex: -a)
  • Binários — agem sobre dois operandos (ex: a + b)

Operadores Unários

O operador unário mais comum é o -, que inverte o sinal de um número: 

a = -10  // a agora é -10

Precedência dos Operadores

A precedência define a ordem em que as operações em uma expressão são executadas.

  1. Parênteses ()
  2. Unário negativo -
  3. Potência **
  4. Multiplicação, Divisão, Módulo *, /, %
  5. Adição e Subtração +, -

Exemplo:

Resultado = 10 / 2 * 5
// 10 / 2 = 5
// 5 * 5 = 25

Operador de Atribuição

O operador = é usado para atribuir um valor a uma variável: 

x = 10
y = x + 5  // y agora vale 15

Operadores Relacionais (Comparação)

Operadores relacionais comparam dois valores e retornam um valor true (verdadeiro) ou false (falso).

OperadorOperaçãoExemploResultado
<Menor que3 < 5true
>Maior que5 > 3true
==Igual a5 == 5true
!=Diferente de5 != 3true
<=Menor ou igual a3 <= 5true
>=Maior ou igual a5 >= 5true

Comparações de strings (cadeias de caracteres) respeitam maiúsculas e minúsculas: 

str1 = "abc"
str2 = "ABC"

str1 == str2  // false

Operadores Lógicos

Operador de Concatenação de Strings

O operador + pode ser usado para concatenar (juntar) strings: 

str1 = "Olá, "
str2 = "mundo!"
resultado = str1 + str2  // "Olá, mundo!"

Exemplos Completos

Exemplo com operadores mistos:

Resultado = (5 + 3) * 2 > 10 && (7 % 2 == 1)

// Passo a passo:
// (5 + 3) = 8
// 8 * 2 = 16
// 16 > 10 = true
// 7 % 2 = 1 → 1 == 1 → true
// true && true = true

Resumo

  • Operadores aritméticos realizam cálculos matemáticos (+, -, *, /, %, **).
  • Operadores relacionais comparam valores e retornam true ou false (==, !=, <, >, <=, >=).
  • Operadores lógicos combinam condições (&&, ||, !).
  • A precedência dos operadores determina a ordem de execução das operações.
  • Operadores de concatenação + unem strings.
  • Operadores de atribuição = armazenam valores em variáveis.
