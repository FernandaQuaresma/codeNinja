Vetores e Matrizes

Introdução

Quando precisamos armazenar **vários valores relacionados**, como uma lista de números, nomes ou notas, utilizamos vetores e matrizes.

Essas estruturas permitem organizar os dados de forma eficiente e acessar qualquer elemento de maneira rápida, utilizando um **índice** (posição).

Vetores (Arrays)

Um vetor é uma estrutura que armazena uma sequência de valores do mesmo tipo, dispostos em uma linha (estrutura unidimensional).

Cada elemento do vetor é identificado por um **índice**. Na maioria das linguagens, o índice começa em 0.

Sintaxe de declaração de vetores

int v[5]; // vetor de 5 inteiros em C String nomes[10]; // vetor de 10 strings em Java

Acesso aos elementos

Para acessar um elemento usamos:

v[0]; // primeiro elemento v[4]; // quinto elemento (índice 4)

Percorrer vetores com laços

Geralmente utilizamos um for para percorrer os elementos:

for (int i = 0; i < 5; i++) { printf("%d\n", v[i]); }

Matrizes (Arrays multidimensionais)

Uma matriz é um **array de duas dimensões**, organizada em **linhas e colunas**.

Cada elemento é acessado com dois índices: linha e coluna.

Sintaxe de declaração de matrizes

int m[3][4]; // matriz de 3 linhas e 4 colunas

Acesso aos elementos da matriz

m[0][0]; // 1ª linha, 1ª coluna m[2][3]; // 3ª linha, 4ª coluna

Percorrer matrizes com laços

Utilizamos dois laços for aninhados:

for (int i = 0; i < 3; i++) { for (int j = 0; j < 4; j++) { printf("%d ", m[i][j]); } printf("\n"); }

Boas práticas

  • Inicialize sempre o vetor ou matriz antes de usar.
  • Verifique sempre os limites de índice para evitar erros de acesso inválido.
  • Utilize nomes descritivos para seus vetores e matrizes.
  • Lembre-se que o último índice válido é tamanho - 1.

Exemplos práticos

Exemplo 1 - Preencher e exibir um vetor

int v[5]; for (int i = 0; i < 5; i++) { v[i] = i + 1; } for (int i = 0; i < 5; i++) { printf("v[%d] = %d\n", i, v[i]); }

Exemplo 2 - Preencher uma matriz com valores crescentes

int m[3][3]; int valor = 1; for (int i = 0; i < 3; i++) { for (int j = 0; j < 3; j++) { m[i][j] = valor++; } } for (int i = 0; i < 3; i++) { for (int j = 0; j < 3; j++) { printf("%d ", m[i][j]); } printf("\n"); }

Resumo

  • Vetores: armazenam uma sequência de valores do mesmo tipo.
  • Índices começam em 0.
  • Utilizamos for para percorrer vetores.
  • Matrizes: armazenam valores em formato de tabela (linhas e colunas).
  • Utilizamos **dois laços for aninhados** para percorrer uma matriz.
  • Evite **acesso fora dos limites** (gera comportamento indefinido ou erro).
